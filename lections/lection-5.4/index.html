    Lection 5.4

    1. Запрос на статический сайт

    Для начала, давайте отправим GET-запрос на обычный веб-сайт, чтобы получить HTML-контент страницы. Мы будем использовать класс HttpClient для отправки запроса и получения ответа.

    Пример кода:

    using System;
using System.Net.Http;
using System.Threading.Tasks;

class Program
{
    static async Task Main(string[] args)
    {
        // Указываем URL сайта, с которого будем получать контент
        string url = "https://www.example.com";

        // Создаем объект HttpClient
        using (HttpClient client = new HttpClient())
        {
            try
            {
                // Отправляем GET-запрос и ждем ответа
                HttpResponseMessage response = await client.GetAsync(url);

                // Проверяем, успешен ли запрос
                if (response.IsSuccessStatusCode)
                {
                    // Читаем содержимое ответа как строку
                    string content = await response.Content.ReadAsStringAsync();

                    // Выводим содержимое страницы
                    Console.WriteLine(content);
                }
                else
                {
                    Console.WriteLine("Ошибка при запросе. Статус: " + response.StatusCode);
                }
            }
            catch (Exception ex)
            {
                Console.WriteLine("Произошла ошибка: " + ex.Message);
            }
        }
    }
}

    Объяснение:

    1. HttpClient — используется для отправки HTTP-запросов.
    2. GetAsync(url) — отправляет асинхронный GET-запрос на указанный URL.
    3. response.IsSuccessStatusCode — проверяет, был ли запрос успешным (статус 2xx).
    4. response.Content.ReadAsStringAsync() — извлекает тело ответа как строку (в данном случае HTML-контент страницы).
    5. Если запрос не успешен, выводим код ошибки.

    2. Запрос к API

    Теперь давайте сделаем запрос к API, чтобы получить данные в формате JSON. В качестве примера возьмем публичное API, например, для получения информации о погоде с OpenWeatherMap.

    Пример кода для запроса к API:

    using System;
using System.Net.Http;
using System.Threading.Tasks;

class Program
{
    static async Task Main(string[] args)
    {
        // Ваш API-ключ для OpenWeatherMap
        string apiKey = "ваш_ключ_от_openweathermap";
        // Город, для которого будем получать погоду
        string city = "Moscow";
        // URL для запроса
        string url = $"http://api.openweathermap.org/data/2.5/weather?q={city}&appid={apiKey}";

        // Создаем объект HttpClient
        using (HttpClient client = new HttpClient())
        {
            try
            {
                // Отправляем GET-запрос к API
                HttpResponseMessage response = await client.GetAsync(url);

                // Проверяем успешность запроса
                if (response.IsSuccessStatusCode)
                {
                    // Читаем ответ как строку (в формате JSON)
                    string jsonResponse = await response.Content.ReadAsStringAsync();

                    // Выводим полученный JSON (для демонстрации)
                    Console.WriteLine("Ответ от API:");
                    Console.WriteLine(jsonResponse);

                    // Вы можете использовать JSON-библиотеку для десериализации (например, Newtonsoft.Json)
                    // В этом примере просто выводим весь JSON как строку.
                }
                else
                {
                    Console.WriteLine("Ошибка при запросе. Статус: " + response.StatusCode);
                }
            }
            catch (Exception ex)
            {
                Console.WriteLine("Произошла ошибка: " + ex.Message);
            }
        }
    }
}

    Объяснение:

    1. В коде мы подставляем API-ключ (который можно получить, зарегистрировавшись на OpenWeatherMap).
    2. Формируем URL для запроса к API с использованием названия города и ключа.
    3. Отправляем GET-запрос с помощью client.GetAsync(url).
    4. Проверяем, успешен ли запрос с помощью response.IsSuccessStatusCode.
    5. Извлекаем ответ в формате JSON и выводим его на консоль с помощью response.Content.ReadAsStringAsync().

    3. Как читать результаты

    В случае с статическим сайтом, результат — это HTML-контент страницы, который можно просто вывести на консоль.

    Пример:

    <html>
<head>
  <title>Example Domain</title>
</head>
<body>
  <h1>Example Domain</h1>
  <p>This domain is established to be used for illustrative examples in documents.</p>
</body>
</html>

    В случае с API, результат будет в формате JSON. Например, для запроса к OpenWeatherMap API вы получите ответ в таком виде:

    {
  "coord": { "lon": 37.6173, "lat": 55.7558 },
  "weather": [
    {
      "id": 800,
      "main": "Clear",
      "description": "clear sky",
      "icon": "01d"
    }
  ],
  "base": "stations",
  "main": {
    "temp": 296.69,
    "feels_like": 295.52,
    "temp_min": 296.69,
    "temp_max": 296.69,
    "pressure": 1015,
    "humidity": 51
  },
  "visibility": 10000,
  "wind": {
    "speed": 3.09,
    "deg": 101
  },
  "clouds": { "all": 0 },
  "dt": 1617184065,
  "sys": {
    "type": 1,
    "id": 9026,
    "country": "RU",
    "sunrise": 1617183125,
    "sunset": 1617230529
  },
  "timezone": 10800,
  "id": 524901,
  "name": "Moscow",

}

    Для чтения этого JSON-ответа и извлечения нужной информации (например, температуры), вам нужно десериализовать его в объект. Для этого в C# можно использовать библиотеку Newtonsoft.Json.

    Пример десериализации JSON:

    using Newtonsoft.Json;

var weatherData = JsonConvert.DeserializeObject<dynamic>(jsonResponse);
Console.WriteLine("Температура: " + weatherData.main.temp);

    Этот код извлекает температуру из полученного JSON-ответа.

    Итог

    1. HttpClient — основной инструмент для отправки HTTP-запросов в C#.
    2. GET-запросы используются для получения данных с веб-сайтов или API.
    3. Ответы могут быть в виде HTML (страница сайта) или JSON (данные с API).
    4. Для работы с JSON в C# используется библиотека Newtonsoft.Json, которая позволяет легко преобразовывать строки JSON в объекты и обратно.
