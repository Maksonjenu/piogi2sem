Лабораторная работа 3

Вступление

Эта лабораторная работа посвящена разработке системы управления комнатой "Умный дом" с использованием C#. Основная цель - продемонстрировать принципы объектно-ориентированного программирования, такие как классы, свойства, модификаторы доступа, события и методы.

Классы, события, свойства

Создание системы управления комнатой с возможностью контроля освещения и температуры.

Задание 1. Создание класса с конструктором и свойствами

Задачи:

  1. Создать класс Room, включающий в себя:

    • Поле name комнаты;
    • Поле temperature комнаты;
    • Свойством Name комнаты только для чтения (get).
    • Свойством Temperature — температура в комнате (public get; private set; ).

  2. Реализовать конструктор,    принимающий     строковый параметр для инициализации свойства «Названия комнаты». Реализовать проверку корректности значения (не пустая строка).

  3. В       главном      окне  (программе) создать     объект класса      (например, с именем «Кухня») и вывести его имя и температуру в TextBlock. 

string.IsNullOrWhiteSpace("строка") // – возвращает true или false.

Задание 2. Использование модификаторов доступа

Задачи:

  1. Добавить приватное поле в класс «Комната», отвечающее за состояние света (например, bool isLighOn).

  2. Реализовать публичное        свойство, которое        предоставляет     доступ к полю состояния света (bool IsLightOn).

  3. Реализовать публичное        свойство, которое        предоставляет     строковое представление состояния (например, string LightState).

  4. В главном окне напрямую изменить isLightOn и IsLightOn, чтобы продемонстрировать работу модификаторов доступа.

Задание 3. Реализация методов для управления состоянием

Задачи:

  1. В классе «Комната» реализовать методы:

    • Публичный метод,        который     «включает  свет» (пример: TurnLightOn()).
    • Публичный метод,        который     «выключает         свет» (пример: TurnLightOff()).
    • Публичный метод, который изменяет значение свойства Temperature (пример: SetTemperature(int newTemperature)).

  2. В главном окне протестировать эти методы, изменяя состояние света и температуры, и выводить результаты в текстовый блок.

Дополнительно: добавить проверку в SetTemperature, чтобы температура была в диапазоне от 15 до 30 градусов.

Задание 4. Введение событий для отслеживания изменений состояния Задачи:

  1. В классе «Комната» объявить публичное событие изменения состоянияосвещения (пример: public event Action<string> OnLightStateChanged).
  2. В       методах включения      и       выключения        света вызывать OnLightStateChanged, передавая сообщение о новом состоянии света или использовать свойство состояния света.
  3. В главном окне подписаться на событие OnLightStateChanged и выводить полученное сообщение в элемент интерфейса.
  4. Аналогично реализовать событие для изменения температуры.

Задание 5. Добавление метода для получения информации о комнате

Задачи:

  1. В классе «Комната» создать метод public string GetRoomInfo(), который возвращает строку с информацией о названии комнаты, состоянии света и текущей температуре.
  2. В главном окне вызвать GetRoomInfo() и вывести результат на экран.

Задание 6: Добавление элементов управления Задачи:

В главном окне реализовать элементы управления:

  1. Состоянием света (CheckBox или RadioButton);
  2. Температурой (Slider);
  3. Название комнаты (TextBlock или ComboBox).
