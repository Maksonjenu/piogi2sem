Лабораторная работа 1

Вступление

Целью этой работы будет получение базовых навыков программирования на языке высокого уровня C# в среде программирования Microsoft Visual Studio.

Задания:

В рамках данной лабораторной работы предлагается разработка графических интерфейсов пользовательских приложений.

1. "Калькулятор"

  • Задача: Разработка графического интерфейса приложения "Калькулятор".

Приложение должно имитировать функциональность стандартного калькулятора, позволяя пользователю выполнять базовые арифметические операции.

2. "Календарь"

  • Задача: Разработка графического интерфейса приложения "Календарь".
  • Ожидаемое поведение: Приложение должно отображать календарь, позволяя пользователю переходить между месяцами и годами с использованием цветов для выделения выходных дней.

3. "Погодный Виджет"

  • Задача: Разработка графического интерфейса приложения "Погодный виджет".
  • Ожидаемые возможности:
    • Отображение текущей температуры.
    • Отображение названия города.
    • Отображение иконки, соответствующей текущей погоде.
