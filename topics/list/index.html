    List

    Что такое список List.

    List— это коллекция, которая позволяет хранить элементы в динамическом порядке. Она является оберткой вокруг массива, но с возможностью автоматического расширения по мере добавления элементов.

    Это структура данных, которая значительно облегчает работу с динамическими набором данных.

    1. Создание списка

    Для создания списка в C# используется класс List<T>, где T — это тип данных, который будет храниться в списке. Например, чтобы создать список целых чисел:

    List<int> numbers = new List<int>();

    Можно также инициализировать список значениями сразу:

    List<int> numbers = new List<int> { 1, 2, 3, 4, 5 };

    2. Добавление элементов в список

    Чтобы добавить элементы в список, используется метод Add(). Он добавляет один элемент в конец списка:

    numbers.Add(6);  //Добавляем число 6 в конец списка

    Можно также добавить несколько элементов:

    numbers.AddRange(new int[] { 7, 8, 9 });

    Также в список можно добавить объект пользовательского класса. 

    //Создаем список объектов типа Room.
List<Room> rooms = new List<Room>();

//Создаем новый объект типа Room с именем "Kitchen" и температурой 15.
Room room_1 = new Room("Kitchen",15);

//Добавляем созданный объект room_1 в список rooms.
rooms.Add(room_1);

    или

    //Создаем список объектов типа Room.
List<Room> rooms = new List<Room>();

//Добавляем новый объект типа Room с именем "Kitchen" и температурой 15 в список rooms.
// Создание объекта происходит непосредственно в методе Add, что является сокращенной формой записи.
rooms.Add(new Room("Kitchen", 15));

    Оба подхода добавляют объект Room в список rooms, но делают это немного по-разному.

    Первый подход сначала создает объект Room и сохраняет его в переменной room_1.
    Это позволяет вам работать с объеком до его добавления в список, например, изменять его свойства или использовать его для других операций.

    Второй подход создает объект Room непосредственно в методе Add. Это более краткая запись, особенно если вам не нужно выполнять какие-либо действия с объектом до его добавления в список.

    3. Доступ к элементам списка

    Чтобы получить доступ к элементу списка, можно использовать индекс:

    int firstNumber = numbers[0];  // Получаем первый элемент

    Если нужно изменить элемент:

    numbers[0] = 10;  // Изменяем первый элемент на 10

    4. Удаление элементов из списка

    Удаление элемента можно сделать с помощью метода Remove() для удаления первого найденного элемента с заданным значением:

    numbers.Remove(3);  // Удаляет первое вхождение числа 3

    Если нужно удалить элемент по индексу, используется метод RemoveAt():

    numbers.RemoveAt(2);  // Удаляет элемент по индексу 2 (третью позицию)

    Чтобы удалить все элементы списка, можно использовать метод Clear():

    numbers.Clear();  // Очищает весь список

    5. Поиск в списке

    Для поиска элемента в списке можно использовать методы Contains() и IndexOf().

    • Contains(): проверяет, существует ли элемент в списке:
    bool hasThree = numbers.Contains(3);  // Возвращает true, если 3 есть в списке
    • IndexOf(): возвращает индекс первого вхождения элемента или -1, если элемент не найден:
    int index = numbers.IndexOf(4);  // Возвращает индекс первого вхождения числа 4

    6. Изменение порядка элементов

    Для сортировки списка используется метод Sort():

    numbers.Sort();  // Сортирует список в порядке возрастания

    Чтобы отсортировать список в убывающем порядке, можно использовать перегрузку метода Sort() с компаратором:

    numbers.Sort((a, b) => b.CompareTo(a));  // Сортировка по убыванию

    7. Перебор элементов списка

    Для перебора элементов списка можно использовать цикл foreach:

    foreach (int number in numbers) {
    Console.WriteLine(number);  // Вывод каждого элемента списка
}

    Можно также использовать цикл for с индексом:

    for (int i = 0; i < numbers.Count; i++) {
    Console.WriteLine(numbers[i]);  // Вывод по индексу
}

    8. Линейные методы для работы с элементами

    C# предоставляет множество встроенных методов для работы с элементами списка через LINQ (Language Integrated Query).

    • Where(): фильтрация элементов по условию:
    var evenNumbers = numbers.Where(n => n % 2 == 0).ToList();
    • Select(): преобразование элементов:
    var doubledNumbers = numbers.Select(n => n * 2).ToList();
    • OrderBy(): сортировка списка:
    var sortedNumbers = numbers.OrderBy(n => n).ToList();

    9. Пример работы с List<T>

    using System;
using System.Collections.Generic;

class Program {
    static void Main() {
        // Создаем список
        List<int> numbers = new List<int> { 1, 2, 3, 4, 5 };

        // Добавляем элементы
        numbers.Add(6);
        numbers.AddRange(new int[] { 7, 8, 9 });

        // Печатаем все элементы списка
        Console.WriteLine("Список:");
        foreach (int number in numbers) {
            Console.WriteLine(number);
        }

        // Удаляем элемент
        numbers.Remove(3);
        Console.WriteLine("\nПосле удаления 3:");
        foreach (int number in numbers) {
            Console.WriteLine(number);
        }

        // Сортируем список
        numbers.Sort();
        Console.WriteLine("\nПосле сортировки:");
        foreach (int number in numbers) {
            Console.WriteLine(number);
        }
    }
}
